## [101] Gestionnaire de banque simple

*Notions*
- Types numériques (``int``, ``float``)
- Définition de fonctions simples
- Conditions
- Boucle ``do ... while``
- Variables globales

**Objectif :** Concevoir une "interface console" simple d'une banque qui propose de débiter ou de créditer une somme d'argent dans un compte.
